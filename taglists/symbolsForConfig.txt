Symbols used for config:
Definitions:
    genus: a grouping of related tags, i.e. "Materials", "Integument", etc.


    *{ begins an entity genus
    }* ends an entity genus

    *[ begins an item genus
    ]* ends an item genus

    # denotes a tag line

    $ denotes a comment line

    | denotes a separration of an element in a line


Tag Formatting
All fields are given as modifiers.

*[
    # Name|weightMod
]*
